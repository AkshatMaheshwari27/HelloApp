# Java Command-Line Applications (HelloApp)

A collection of Java-based tools demonstrating command-line argument handling, default value logic, and array iteration. This project follows a feature-branching workflow to manage different functional requirements.

## 🛠️ Features
- **Personalized Greetings**: Processes user input to generate custom messages.
- **Robustness**: Handles cases with no input by providing default values ("World").
- **Bulk Processing**: Support for multiple command-line arguments using array traversal.

## 📂 Branching Strategy
Each major update is developed in a separate branch for clean version control:
- `main`: Documentation and project overview.
- `feature/UCx`: Specific Use Case implementations (UC1 - UC4).
